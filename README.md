## App Description
Phone Caddie is a web app that helps golfers better understand their golf game. User will be able to input shot distances and scores, they will receive suggestions to better improve their game.

## Intended Users
Intened users would be any golfers that are trying to improve their golf game.

## Planned Features
1. Shot input feature
2. Distance tracker and trend analysis
3. Suggestion feedback
